CARD 1: TOTAL PACKETS
  • Cumulative count of all packets analyzed
  • Updates in real-time
  • Color: Neon green

CARD 2: INTRUSIONS
  • Number of intrusions detected
  • Updates immediately when threat detected
  • Color: Neon red

CARD 3: NORMAL TRAFFIC
  • Number of normal packets
  • Updates in real-time
  • Color: Neon green

CARD 4: THROUGHPUT
  • Packets per second (current)
  • Updates every second
  • Color: Cyan

CARD 5: DETECTION RATE
  • Percentage of intrusions
  • Calculated as: (Intrusions / Total Packets) * 100
  • Color: Yellow

CARD 6: AVG LATENCY
  • Average prediction latency in milliseconds
  • Shows system performance
  • Color: Cyan
